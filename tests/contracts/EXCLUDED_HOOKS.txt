# EXCLUDED_HOOKS.txt
# Intentionally unregistered hooks — each entry must have a reason.
#
# Format: <hook-filename.sh> | <reason>
#
# To whitelist a new unregistered hook, add it here with a reason.
# Do NOT add hooks that should be registered — fix those in .claude/settings.json instead.
#
# Categories:
#   MANUAL_TRIGGER   — invoked on demand, not via Claude event hooks
#   GIT_HOOK         — wired to git, not Claude's hook system
#   LIBRARY          — helper sourced by other hooks, not a hook itself (in _lib/)
#   DEPRECATED       — superseded by another hook; kept for reference
#   FUTURE           — planned for a future sprint; not yet wired
#   CONDITIONAL      — wired conditionally (e.g. only when a service is active)
#   ADMIN            — one-shot admin script, not meant to run on every event
#   INFRA            — infrastructure setup script; run manually or by CI
