# ADR-029 — Anti-reinvention gate (reinvention-check.sh wired)

**Status**: Accepted
**Date**: 2026-04-20
**Deciders**: Maintainer

---

## Context

This session produced `lib/agent_heartbeat.py` (reverted in a prior commit), which
duplicated `lib/agent_bus.py`. Root cause: ADR-028 D1.C was written without grepping
the codebase for existing pub/sub infrastructure. The `reinvention-check.sh` hook
existed on disk with real logic but was never registered in `.claude/settings.json`
(documented in `rules/ROADMAP.md` §1.7 as "exists on disk, not registered").

Because the hook never fired, no gate warned the agent that `agent_bus.py` already
provided Valkey pub/sub capabilities before `agent_heartbeat.py` was drafted.

## Decision

Wire `reinvention-check.sh` as a `PreToolUse:Agent` hook in:

- `scripts/apply-efficiency-profile.sh` — `default` and `full` profiles
- `templates/security-profiles/standard.json` — standard security
  profile (consumed by `set-security-profile.sh standard`)

The hook fires on any sub-agent launch (`Task` tool) whose prompt contains
creation-intent keywords (`create`, `implement`, `write`, `add`) combined with
file-type targets (`lib/`, `hooks/`, `new file`, etc.).

## Scope

### What the hook checks

1. Extracts target filenames from the agent prompt (regex: `[a-z][a-z0-9_-]+\.(sh|py|go|ts|js)`).
2. Runs `find` across `lib/`, `hooks/`, and registered plugin dirs for basename matches.
3. If matches are found: emits a structured warning to stderr listing existing files.
4. Appends a JSON record to `.cognitive-os/metrics/reinvention-checks.jsonl`.

### What it does NOT do (Phase A)

The hook exits 0 regardless — advisory only. It does not hard-block. A hard-block
gate (Phase B) is future work and requires a similarity threshold mechanism.

### Excluded profile

`lean` (now remapped to `default` by ADR-002) is not a separate tier. The hook is
absent from the `minimal` security profile because minimal is reserved for
low-overhead environments.

## Behaviour observed

The hook already contains:

```bash
if ! echo "$PROMPT" | grep -qiE '(create|implement|write|add)'; then exit 0; fi
if ! echo "$PROMPT" | grep -qE '(lib/|hooks/|new file|new hook|new script)'; then exit 0; fi
```

This means it will NOT fire on every agent launch — only on prompts that signal
new-file creation. False-positive rate is low.

## Emit

On every match the hook appends to:
```
.cognitive-os/metrics/reinvention-checks.jsonl
{"timestamp":"…","target":"agent_heartbeat.py","sources":"/path/to/agent_bus.py\n"}
```

No MetricEvent bus call (Valkey is OFF by default). File-based metrics are the
fallback per `rules/observability.md`.

## What it blocks (Phase A vs Phase B)

| Phase | Behaviour |
|-------|-----------|
| A (now) | Advisory: lists existing modules in stderr, logs to JSONL. Exit 0. |
| B (future) | Hard-block when similarity >= 0.7 unless prompt includes `VERDICT: replace\|parallel (rationale)`. |

Phase B requires a semantic similarity check (embeddings or keyword Jaccard). Not
implemented. Track as `sdd/reinvention-gate-phase-b`.

All verified passing at time of commit.

## Integration

- Complements `rules/reinvention-prevention.md` — that rule was agent-instruction-only
  until this ADR wired the enforcement hook.
- Complements ADR-028b §Lessons ("Discovery Pass mandatory in ADRs") — the hook
  enforces at execution time what the lesson asks of authors at writing time.
- Complements `templates/agent-preamble.md` — preamble tells sub-agents to grep
  before building; this hook provides the automatic backstop when they do not.
- Updates `rules/ROADMAP.md` §1.7 — item is now RESOLVED.

## Rollback

Remove the `hook_entry "reinvention-check.sh"` line from
`scripts/apply-efficiency-profile.sh` and re-run:

```bash
bash scripts/apply-efficiency-profile.sh default
```

The hook file remains on disk (safe). The JSONL metrics file is also left in place.

## Measurable outcome

Track `matches_found` events in `.cognitive-os/metrics/reinvention-checks.jsonl`
over 2 weeks. Expected: >= 1 match per month that would have surfaced a potential
duplicate before it was committed.
