# How to Extend Cognitive OS

> The extension procedures are now implemented as skills. Use the appropriate skill:

| Extension Type | Skill | Command |
|---|---|---|
| Add a hook | `/add-hook` | Guides you through creating and registering a new hook |
| Add a rule | `/add-rule` | Guides you through creating a new rule |
| Add a skill | `/add-skill` | Guides you through creating a new skill |
| Add an MCP server | `/add-mcp` | Guides you through integrating a new MCP server |

For architectural context, see `docs/architecture.md`.
